# Project Template

Opinionated Node.js project template.

Core:
- Framework: Next.js
- Database: PostgreSQL
- ORM: Drizzle ORM
- Auth: Clerk
- Styling: TailwindCSS

Niceties:
- Data validation: Zod
- State management: Zustand
- Environment variables management: t3-env
- Style System: Shadcn
- Formatting: Prettier
- Linting: ESLint

Scripts:
- "feat": Folder structure generation
- "db:push": Schema migration
- "db:studio": Table view
